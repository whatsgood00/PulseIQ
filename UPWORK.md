## Project Description

```
A production-quality analytics dashboard UI built as a frontend portfolio project — inspired by enterprise SaaS products like Mixpanel and Stripe Dashboard.

🔹 What I built
• Full dashboard with revenue, users, conversion & MRR metrics
• Interactive charts (Recharts) with daily / weekly / monthly filters
• Complete auth flow: sign in, sign up, forgot password
• Notifications inbox with read/unread states
• Subscription billing page with 3-tier pricing
• User settings & profile management
• Dark & light theme with zero flash on load
• Fully responsive — desktop, tablet, mobile

🔹 Tech stack
Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS · Framer Motion · Recharts · Zustand · ESLint 9

🔹 Highlights
• Mock REST + GraphQL API layer for realistic data flow
• Reusable component architecture (UI primitives, layouts, pages)
• Production build passes lint & TypeScript checks
• Optimized static generation via Next.js

🔹 Note
Frontend demo with simulated API — showcases UI/UX implementation and modern React patterns. Backend integration ready.

🔗 Source code: https://github.com/whatsgood00/PulseIQ
Run locally: git clone → npm install → npm run dev → http://localhost:3000
```
